JavaScript

JavaScript brings logic and behavior to the web. It is used for conditions, loops, events, arrays, functions, objects, and dynamic programming.

1. Variables and Data Types

JavaScript works with numbers, strings, booleans, arrays, objects, null, and undefined. Variables are commonly declared using let and const.

2. Conditions and Loops

if, else if, else, switch, for, while, and for...of are used to control program flow.

3. Functions

Functions organize logic into reusable blocks. JavaScript supports regular functions, arrow functions, and callback functions.

4. Arrays and Objects

Arrays store ordered data, while objects store key-value pairs. Common array methods include map, filter, and reduce.
