HTML Basics

HTML forms the structure of every webpage. It defines headings, paragraphs, links, forms, images, tables, media, and semantic sections that make content meaningful and accessible.

Semantic Tags Forms Media Elements Accessibility
Back to Dashboard

1. Structure of a Page

A standard HTML document begins with a doctype declaration, followed by the html, head, and body tags. The head contains metadata like title and viewport settings, while the body contains everything visible on the page.

2. Semantic Tags

Semantic tags such as header, nav, section, article, aside, and footer improve readability and accessibility. They help search engines and screen readers understand the meaning of different parts of the page.

  • header for the top area
  • nav for navigation links
  • section for grouped content
  • article for self-contained content

3. Forms and Inputs

Forms allow users to enter and submit data. Common form controls include input, textarea, select, radio buttons, and checkboxes.

4. Media and Accessibility

HTML supports images, audio, video, and picture elements. Accessibility is improved by using alt text, labels, correct heading order, and meaningful link text.
