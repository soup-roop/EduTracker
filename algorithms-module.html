Algorithms

Algorithms solve problems step by step. Their quality is judged by correctness, efficiency, and how well they scale with larger inputs.

1. Searching

Linear search checks items one by one, while binary search repeatedly halves the search range when the input is sorted.

2. Sorting

Common sorting algorithms include bubble sort, selection sort, insertion sort, merge sort, and quick sort.

3. Complexity

Big-O, Omega, and Theta notation describe worst-case, best-case, and tight-bound growth of algorithms.

4. Choosing the Right Algorithm

Efficient algorithm choice depends on input size, order of data, memory limits, and required performance.
