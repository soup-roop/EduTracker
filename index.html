DSA + FWD Project

A multi-page educational tracker with live analytics.

Open modules on separate pages, read content, bookmark topics, mark them complete, and watch the dashboard update automatically using browser storage.

Core Concepts Used

  • Queue for event tracking
  • Stack for undo
  • Map-style stats storage
  • Sorting for top topics
  • Search for modules
  • Linked-list style recent activity
